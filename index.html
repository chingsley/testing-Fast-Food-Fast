Welcome to fast-food-fast api.

OUR RESOURCES
url resource (in json format)
GET /api/v1/orders fetches all the orders
GET /api/v1/orders/[orderID] fetches a particular order of orderID
POST /api/v1/orders to add a new order
PUT /api/v1/orders/[orderID] updates the status of a particular order of orderID
DELETE /api/v1/orders/[orderID] deletes the status of a particular order of orderID
GET /api/v1/items fetches all the items
GET /api/v1/items/[itemID] fetches a particular item of itemID
POST /api/v1/items to add a new item
PUT /api/v1/items/[itemID] updates the status of a particular item of itemID
DELETE /api/v1/items/[itemID] deletes the status of a particular item of itemID
